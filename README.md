# Alexandria Podcast

Marcelův osobní podcast feed s denními audio dávkami AI/Trading/Longevity/PhD research scanů z Alexandria knowledge base.

**Apple Podcasts subscription URL**:
https://marcelv3612.github.io/alexandria-podcast/feed.xml

V Apple Podcasts: Library → ... → Follow a Show by URL.

Audio se generuje denně z `alexandria/wiki/research/auto/hot/` pomocí `daily-audio.sh` (Edge TTS cs-CZ-AntoninNeural) a publikuje jako GitHub Release přes `build-podcast-feed.sh`.

Repo není určen ke klonování ostatními — feed.xml a release assety slouží jako podcast distribuce.
